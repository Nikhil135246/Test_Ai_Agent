# Test_Ai_Agent

A small, local web project demonstrating a simple static site with HTML, CSS, and JavaScript.

## What it is

This repository contains a minimal frontend project—an `index.html` page with styling in `style.css` and behavior in `script.js`. It's intended as a playground for experimenting with DOM manipulation and basic UI layout.

## Features

- Single-page static site
- Simple JavaScript interactions (see `script.js`)
- Lightweight and easy to extend

## Files

- `index.html` — main page
- `style.css` — styles
- `script.js` — JavaScript behavior

## Usage

1. Open `index.html` in your browser to view the site.
2. For development, use a local dev server (recommended) to enable live reload and avoid CORS issues. Example using Python 3:

```
python -m http.server 8000

# then open http://localhost:8000 in your browser
```

## Development

- Edit `index.html`, `style.css`, or `script.js` and refresh the browser.
- Consider installing a VS Code extension like Live Server for instant reload.
